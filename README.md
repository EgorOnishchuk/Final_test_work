 # Описание проекта
## Структура проекта
1. Программа.
2. Блок-схема ключевой функции программы.
3. Характеристика проекта.
## Описание программы
Программа, представленная в данном проекте, принимает на вход посредством ввода пользователем данных с помощью системной консоли элементы массива, имеющие строковый тип данных, и выводит те из них, которые имеют не более 3 символов. Для реализации алгоритма использовался 1 массив. Программа написана на языке программирования C#.
